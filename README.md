# CS-350
Embedded systems and emerging technologies portfolio projects using Raspberry Pi and Python.

CS 350 Portfolio Reflection
Project Overview
In this course, I worked on multiple embedded systems and emerging technology projects using Raspberry Pi hardware and Python programming. These projects focused on hardware and software integration, GPIO communication, serial communication, temperature monitoring, and system automation.
The two projects I selected for this portfolio are the Thermostat Project and the Serial Light Control Project. These projects demonstrate my ability to interface software with hardware components and troubleshoot real-world embedded systems issues.
Thermostat Project
Problem the Project Was Solving
The thermostat project was designed to simulate a smart environmental control system using a Raspberry Pi, temperature sensor, and LEDs. The system monitored room temperature and used LEDs to indicate heating and cooling states based on a set temperature value.
What I Did Well
I successfully integrated hardware sensors with Python software logic and implemented LED state changes based on environmental input. I also improved my troubleshooting skills while working with GPIO pins, wiring, and sensor communication.
Where I Could Improve
I could improve code optimization, hardware organization, and documentation. In the future, I would like to improve the user interface and add more advanced automation features.
